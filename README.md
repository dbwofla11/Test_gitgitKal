게임 시작 화면

"공대 탈출 일지"는 학교를 배경으로 평소 학교 생활 중 느끼는 탈출 욕구를 해소하고자 개발되었습니다. 이 게임은 친숙한 학교 공간과 괴담을 창의적으로 결합하여 플레이어의 호기심을 자극하며, 함정과 몬스터에 대한 긴장감을 통해 감정적으로 강렬한 경험을 제공합니다. 이를 통해 플레이어는 학교 생활을 다양한 시각에서 탐험하고 새로운 재미와 스릴을 느낄 수 있습니다.
